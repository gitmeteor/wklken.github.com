拿到一台干净的centos之后, 初始化Python环境, 一些命令和问题记录而已

可以搞成脚本自动初始化, 当然, 用docker更好

基础环境

1. 创建用户

```bash sudo adduser newuser sudo passwd newuser

设置授权不需要输入密码

sudo /usr/sbin/visudo newuser ALL=NOPASSWD: ALL ```

2. EPEL(Fedora Extra Packages for Enterprise Linux repository )

参考文档

bash wget http://dl.fedoraproject.org/pub/epel/6/x86_64/epel-release-6-8.noarch.rpm sudo rpm -Uvh epel-release-6*.rpm

3. 加ius源

包地址

bash wget http://dl.iuscommunity.org/pub/ius/stable/CentOS/6/x86_64/ius-release-1.0-13.ius.centos6.noarch.rpm sudo rpm -Uvh ius-release-1.0-13.ius.centos6.noarch.rpm

4. 安装python2.7 / python3 / pip

bash sudo yum install python27 sudo yum install python27-devel

then

```bash curl https://bitbucket.org/pypa/setuptools/raw/bootstrap/ez_setup.py | sudo /usr/bin/python2.7 -

pip

curl https://raw.githubusercontent.com/pypa/pip/master/contrib/get-pip.py | sudo /usr/bin/python2.7 - ```

extra: install python3

bash sudo yum install python34u python34u-devel

5. virtualenv

bash sudo pip install virtualenv

6. gen ssh key

```bash ssh-keygen -t rsa

add ~/.ssh/id_rsa.pub to git or github ```

一些服务

1. install git

bash sudo yum install git

2. mysql

bash sudo yum install mysql sudo yum install mysql-devel* -y sudo yum install mysql-server sudo /sbin/service mysqld start

3. redis

bash sudo yum install redis

4. rabbitmq

bash sudo yum install rabbitmq-server

问题

[Errno 14] Peer cert cannot be verified or peer cert invalid

add "sslverify=false" in /etc/yum.conf

pip error: ImportError: cannot import name HTTPSHandler

bash yum install openssl openssl-devel -y

see in sof
